Componentes técnicos esperados:

Chatbot conversacional com contexto clínico injetado via system prompt e memória de sessão.

RAG (Retrieval-Augmented Generation) sobre base de conhecimento clínico e políticas Care Plus.

Function Calling para consulta simulada a histórico do paciente, verificação de interações medicamentosas e agendamentos.

Arquitetura multi-agente (opcional, bônus) usando LangGraph para orquestrar check-up, triagem e prescrição.

Suite de avaliação automatizada (evals) para medir qualidade das respostas.
